検索とカタログ表示の動作をコントロールする為に Search Settings を利用します。Settings モジュールの "Search" タブを開きます。ここで、ストアフロントと Virto Commerce  Manager の為の Search Settings を行います。インデックスを再構築するには、"REBUILD" ボタンをクリックします。
